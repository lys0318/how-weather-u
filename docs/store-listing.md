## 자세한 설명 (4000자 이내)

```
🌤️ 오늘의 날씨가 당신에게 건네는 한마디

하우웨더유는 현재 위치의 날씨와 시간대에 맞춰
AI가 위로·응원·조언 메시지를 들려드리는 감성 앱이에요.
하루를 좀 더 따뜻하게 시작하고 싶은 분께 추천드려요.


✨ 무엇을 받을 수 있나요?

▶ 💖 위로 메시지
   힘든 하루를 보내는 당신에게 따뜻한 공감을 건네요.

▶ 🔥 응원 메시지
   에너지를 북돋아주는 격려와 활기찬 한마디.

▶ 💡 조언 메시지
   날씨를 활용해 "지금 해보면 좋을 일"을 추천해줘요.
   "비 오는 날엔 미루던 일을 차분히 정리해보세요" 같은 식으로.

▶ 🌈 오늘의 활동 추천
   "비 오는 날엔 집에서 책 한 권 어떠세요?"
   "햇빛 좋은 오늘은 친구랑 농구 한 게임!"
   날씨에 어울리는 활동을 친구처럼 권해드려요.

▶ 🍱 오늘의 음식 추천
   "비 오는 날엔 김치전에 막걸리"
   "더운 날엔 시원한 냉면 한 그릇"
   현재 기온·시간대에 딱 맞는 음식 추천.


🌅 사용 방법은 간단해요

1. 구글 계정으로 1초 로그인
2. 위치/알림 권한 허용
3. 받고 싶은 카테고리 버튼 탭
4. 끝! 메시지가 도착해요

저장한 메시지는 ★ 북마크해두면 영구 보관돼요.
일반 메시지는 7일간 자동 보관 후 삭제됩니다.


🔔 적당한 알림, 부담 없는 사용

- 아침/점심/오후/저녁에 적절히 알림이 도착해요.
- 새벽 1시 ~ 오전 6시는 방해금지 시간.
- 알림은 메시지를 미리 만들지 않아요. "지금 받아보세요" 식으로 살짝 안내만 해드려요.


🌙 시간대에 따라 달라지는 분위기

낮에는 밝은 하늘색 그라디언트,
밤에는 깊은 미드나잇 톤으로 자동 전환돼요.
날씨에 따라서도 배경이 살짝 바뀌어요.


🔒 개인정보 보호

- 위치는 날씨 조회 시점에만 사용하고 저장하지 않아요.
- 모든 통신은 HTTPS 암호화.
- 앱 내 [설정 → 계정 탈퇴] 한 번으로 모든 데이터 즉시 영구 삭제.
- 개인정보처리방침: https://lys0318.github.io/how-weather-u/privacy-policy.html


💰 무료로 즐기세요

- 메시지·활동·음식 합쳐서 하루 5번까지 무료 이용.
- 매 호출 시 짧은 광고 한 번 시청.
- 추가 결제 옵션은 추후 업데이트 예정.


🤖 안내사항

- 메시지는 Anthropic Claude AI가 생성합니다.
- 날씨 정보는 OpenWeatherMap에서 가져옵니다.
- 본 앱은 의료·심리 상담을 대체하지 않습니다. 깊은 우울감이 지속된다면 전문가와 상담해주세요.


💌 피드백 환영

문의/제안: tosca0318@gmail.com

오늘 하루도 수고 많으셨어요.
하우웨더유가 짧게나마 마음에 닿는 한마디를 건네드릴게요. 🌤️
```
