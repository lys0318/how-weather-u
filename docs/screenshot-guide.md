# 📸 Play Store 스크린샷 가이드

## 규격
- **세로(권장)**: 1080×1920, 1080×2340 (지금 폰 비율) — Play Store가 자동 정렬해줌
- **개수**: 최소 2장 / 최대 8장 / **권장 5~6장**
- **포맷**: JPG 또는 24-bit PNG (알파 X)

## 찍는 방법 (Android)
- 폰에서 화면을 띄운 상태로 **전원 + 볼륨 다운 버튼 동시 누르기**
- 갤러리 → 공유 → PC로 전송 (카카오톡 나에게 보내기 추천)

---

## 📋 찍을 화면 5장 (순서대로)

### 1️⃣ 로그인 화면 ⭐ (대표 이미지)
- 앱 실행 후 첫 화면
- 핵심: "구글로 시작하기" 버튼 + 새벽 그라디언트 배경 + 앱 이름
- **캡션 (편집 추가)**: "오늘의 날씨가 건네는 한마디"

### 2️⃣ 홈 화면 — 날씨만 표시
- 로그인 후 메시지/활동/음식 생성 전 깨끗한 상태
- 핵심: 큰 온도, 날씨 이모지, 도시명, 3개 버튼
- **캡션**: "현재 날씨에 맞춰 AI가 추천해드려요"

### 3️⃣ 메시지 유형 선택 모달
- "오늘의 메시지 받기" 탭 후 떠오르는 위로/응원/조언 선택 화면
- 핵심: 3가지 톤 옵션 가시화
- **캡션**: "기분에 맞는 톤을 골라보세요"

### 4️⃣ 메시지 + 활동 + 음식 카드가 모두 채워진 홈
- 메시지·활동·음식 한 번씩 받아서 카드 3개가 보이는 상태
- 핵심: 실제 AI 답변 콘텐츠 보여주기
- **캡션**: "위로 · 활동 · 음식, 한 번에"

### 5️⃣ 히스토리 화면 — 메시지 기록
- 메시지 기록 탭, 여러 메시지가 쌓여 있는 상태
- 북마크된 메시지가 있으면 더 좋음
- 핵심: 시간순 정리, 북마크 표시
- **캡션**: "기억하고 싶은 한마디는 ★ 북마크로 영구 보관"

### (선택) 6️⃣ 설정 화면
- 사용자 프로필 카드 + 알림 안내가 보이는 화면
- 핵심: 깨끗하고 단순한 UI
- **캡션**: "방해 없이, 적당한 시간에만 알림"

## 💡 찍기 전 팁

1. **상태바 정리**
   - 알림 없는 상태로 (다 읽기 + 진동/무음 일시적 끄기)
   - 배터리 적당히 충전 (50%+ 권장)
   - 시간을 깔끔한 시간대에 (예: 18:00 같은 정각)

2. **앱 데이터 깔끔하게**
   - 광고 차단 (현재 빌드엔 광고 없음)
   - 메시지 몇 개 미리 생성해서 카드 채워두기

3. **실제 콘텐츠로**
   - "안녕하세요" 같은 placeholder 말고, **진짜 생성된 AI 메시지** 보여주기
   - 한국어 문장으로
