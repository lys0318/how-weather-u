4. 보유 및 이용 기간

  • 계정 정보: 회원 탈퇴 시까지. 탈퇴 시 앱 내 [계정 탈퇴] 버튼을 누르면 즉시 영구 삭제됩니다.
  • 사용 기록(usage_log): 수집일로부터 90일 후 자동 삭제. 단, 회원 탈퇴 시 계정 삭제와 함께 즉시 모두 삭제됩니다.
  • 위치 정보: 저장하지 않음 (요청 시점에 1회 사용 후 즉시 폐기)
  • 로컬 메시지 기록: 사용자 기기에 7일간 보관, 이후 자동 삭제 (북마크된 메시지는 영구 보관). 회원 탈퇴 시 폰 내부 데이터도 함께 삭제됩니다.

6. 개인정보 안전성 확보 조치

  • 모든 통신은 HTTPS(TLS 1.2 이상)로 암호화됩니다.
  • API 키 등 민감 정보는 클라이언트 앱에 노출되지 않도록 백엔드 서버(Supabase Edge Functions)에서만 사용됩니다.
  • 사용자 데이터는 Supabase의 Row Level Security(RLS)로 본인만 접근 가능합니다.
  • 관리자 접근은 최소한의 인원으로 제한되며 접근 기록이 남습니다.
